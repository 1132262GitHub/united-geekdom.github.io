26th April 2020

I was not particularly productive on the web today. I managed to add a dark mode to the blog, but I haven't made any progress on my CHARnimation movie. I am working on a pixel art editor with a save code generator on RandomSite's Scratch account, and I'm also working on a CSS code generator at the HTML writer website. I have also redesigned my homepage, and removed the now redundant Updates and Notice Board sections.
In the process of making this blog, I have learned some nifty tricks, mainly in JavaScript and CSS. I use W3 schools for most of my reference code, but where it fails, I use Stack Overflow instead. Most of these tricks actually use a combination of CSS and JavaScript. Below is the code used to make a basic dark mode.
The JS: 
// Use a button to start the function.
function darkMode() {
   var element = document.body;
   element.classList.toggle("dark-mode");
}

The CSS: 
body {
  color: black;
  background-color: white;
}

.dark-mode {
  color: white;
  background-color: black;
}
/*Add more elements to the dark mode by using .dark mode + element. For example, .darkmode h1{color: white;} */

I'll include some more coding tricks in tomorrow's blog post.
For a preview of my CSS code generator, go here for the latest functioning version, or here for the version still in progress. This is all the work I have done today, and will have to pick up the pace tomorrow.
Goodbye from United Geekdom.
